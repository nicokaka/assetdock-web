# Demo Script

## Suggested flow

1. Open the app and sign in with the local seeded admin user.
2. Show the authenticated shell and the main navigation.
3. Open Assets, create a small demo asset, and land on its detail page.
4. Show the asset lifecycle and assignment sections briefly.
5. Open Users and show the list, detail view, and lifecycle controls.
6. Open Audit Logs to show the read-only operational trail.
7. Open Imports to show the CSV import entry point and job viewer.
8. Sign out to close the session flow.

## Demo points

- Cookie-based web session login and logout.
- Authenticated navigation across the main MVP areas.
- Real CRUD foundation for assets and users.
- Lifecycle actions for assets and users.
- Read-only operational visibility through audit logs.
- CSV import entry point already connected to the backend contract.

## Expected duration

3 to 7 minutes.
